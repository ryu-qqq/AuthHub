=== 테넌트 목록 조회

테넌트 목록을 페이징하여 조회합니다.

==== 요청 파라미터

[cols="1,1,1,3"]
|===
|파라미터 |타입 |필수 |설명

|`page`
|Number
|N
|페이지 번호 (기본값: 0)

|`size`
|Number
|N
|페이지 크기 (기본값: 20, 최대: 100)

|`keyword`
|String
|N
|검색 키워드

|`searchField`
|String
|N
|검색 필드 (NAME)

|`status`
|String
|N
|상태 필터 (ACTIVE, INACTIVE)

|`sortKey`
|String
|N
|정렬 키 (CREATED_AT, NAME)

|`sortDirection`
|String
|N
|정렬 방향 (ASC, DESC)
|===

==== 응답 필드

[cols="1,1,3"]
|===
|필드 |타입 |설명

|`data.content`
|Array
|테넌트 목록

|`data.content[].tenantId`
|String
|테넌트 ID

|`data.content[].name`
|String
|테넌트 이름

|`data.content[].status`
|String
|테넌트 상태 (ACTIVE, INACTIVE)

|`data.content[].createdAt`
|String
|생성 일시

|`data.content[].updatedAt`
|String
|수정 일시

|`data.page`
|Number
|현재 페이지

|`data.size`
|Number
|페이지 크기

|`data.totalElements`
|Number
|전체 항목 수

|`data.totalPages`
|Number
|전체 페이지 수
