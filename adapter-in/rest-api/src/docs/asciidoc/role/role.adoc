=== 역할 생성

새로운 역할을 생성합니다.

==== 요청 필드

[cols="1,1,1,2,3"]
|===
|필드 |타입 |필수 |검증 |설명

|`tenantId`
|String
|N
|UUID 형식
|테넌트 ID (null이면 Global 역할)

|`name`
|String
|Y
|2~50자, UPPER_SNAKE_CASE
|역할 이름 (예: USER_MANAGER)

|`displayName`
|String
|N
|최대 100자
|표시 이름 (예: 사용자 관리자)

|`description`
|String
|N
|최대 500자
|역할 설명
|===

NOTE: `name`은 반드시 UPPER_SNAKE_CASE 형식이어야 합니다. (예: `USER_MANAGER`, `ADMIN`, `CONTENT_EDITOR`)

=== 역할 목록 조회

==== 요청 파라미터

[cols="1,1,1,3"]
|===
|파라미터 |타입 |필수 |설명

|`page`
|Number
|N
|페이지 번호 (기본값: 0)

|`size`
|Number
|N
|페이지 크기 (기본값: 20, 최대: 100)

|`tenantId`
|String
|N
|테넌트 ID 필터

|`keyword`
|String
|N
|검색 키워드

|`searchField`
|String
|N
|검색 필드 (NAME, DISPLAY_NAME)

|`types`
|Array
|N
|역할 유형 필터 (SYSTEM, CUSTOM)

|`sortKey`
|String
|N
|정렬 키 (CREATED_AT, NAME)

|`sortDirection`
|String
|N
|정렬 방향 (ASC, DESC)
|===
